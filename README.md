# 🎓 EduMarket МИТУ v2.0

## Локальный запуск

### Backend (Терминал 1)
```
cd backend
npm install
npm run db:init
npm run db:seed   # тестовые данные
npm run dev       → http://localhost:5000
```

### Frontend (Терминал 2)
```
cd frontend
npm install
npm run dev       → http://localhost:3000
```
